## ğŸ“Œ Deskripsi Proyek
  Repositori ini berisi pengumpulan tugas besar mata kuliah Komputasi Statistika yang berfokus pada analisis fluktuasi harga cabai merah besar di Provinsi Lampung menggunakan pendekatan statistik dan komputasi. Dataset yang digunakan diperoleh dari Pusat Informasi Harga Pangan Strategis (PIHPS) Nasional dan mencakup harga harian cabai merah besar pada periode Juli hingga awal Oktober.

  Dalam laporan ini, analisis diarahkan untuk memahami pola fluktuasi harga, mengidentifikasi tingkat ketidakstabilan, serta menilai risiko volatilitas pasar cabai merah besar. Metode yang digunakan mencakup statistika deskriptif, analisis perubahan harga harian, visualisasi tren, serta estimasi ketidakpastian dengan metode bootstrap. Hasil analisis memberikan gambaran menyeluruh mengenai dinamika harga cabai di Lampung dan implikasinya terhadap sektor pangan daerah.

Proses Bootstrap meliputi:
- Menyiapkan data harga harian  
- Membersihkan dan memvalidasi data
- Reshaping data (wide â†’ long)
- Menghitung statistik deksriptif awal
- Menghitung perubahan harga harian 
- Melakukan resamplig bootstrap
- Membentuk distribusi bootstrap
- Menghitung interval kepercayaan 95%
- Mengintrepretasikan ketidakpastian harga
